Why NGX D3 Graphs?
Comfortable
NGX D3 Graphs make it easy to generate D3-based charts by wrapping the code required to construct the entire chart. We don't need to write D3 code any more.
Customizable
NGX D3 Graphs gives some classes to each element when generating, so you can define a custom style by the class and it's possible to extend the structure directly by D3.
Components
NGX D3 Graphs provides just not charts, but also stand-alone graph components such as axes, legend, etc which can be used to create own custom charts and visualizations.
